# Count to 30 to win
Two participants will take turns to chose 1 to 3 numbers that start with the previous number. A goal of this game is who will count to 30 first.
Example:
```
A: 1 2
B: 3 4 5
A: 6
...
A: 27 28 29
B: 30 -> He win
```
